# AnalogRestorator
:ru:

Программа проводит преобразование файла из двоичного формата в формат звука модулированного по частоте

лог. 1 - f

лог. 0 - f/2

**Проект находится в стадии создания прототипа**

**Работоспобность программ не гарантируется**
